## Final project

```sh
sudo apt install cmake libgl1-mesa-dev mesa-common-dev liblzma-dev libvorbis-dev \
  libfreetype6-dev libxrandr2 libxcursor1 libxi6
./run.sh
```
Or (without sound):
```sh
./run-docker.sh
```
